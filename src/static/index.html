NVIDIA RAG Server

Welcome to the NVIDIA RAG Blueprint. This service demonstrates how to set up a RAG solution using NVIDIA NIM and GPU-accelerated components.

Features

  • Document processing with NV-Ingest
  • Vector storage with Pinecone (local or cloud)
  • Question answering powered by NVIDIA NIM

Prerequisites

  • Python 3.12 or later
  • Docker and Docker Compose
  • NVIDIA API key for NIM services
  • Pinecone account (for production deployment)

Quick Start

  1. Configure your environment variables in .env
  2. Start the services with docker compose -f docker-compose.local.yaml up
  3. Upload documents using the API below
  4. Start asking questions!

API Usage

1. Upload Documents

curl -X POST "http://localhost:8000/documents" \
  -H "accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F "files=@your-document.pdf"

2. Ask Questions

curl -X POST "http://localhost:8000/generate" \
  -H "Content-Type: application/json" \
  -d '{"messages": [{"role": "user", "content": "What is...?"}]}'
